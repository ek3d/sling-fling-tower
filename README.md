# Sling Fling Tower
2025.3.14.M Platform Update

## Controls
Click and hold mouse button to aim.  Let go to sling in that direction.

## Powerups
### Rocket
Propels you upwards.

### Multi-Fling
Allows you to sling even if you are in the air.

### Feather
Lowers gravity, allowing you to sling higher.
